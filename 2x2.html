Ortega

Ortega consists of 3 steps:

  1. OFL (Orientate First Layer)
  2. OLL (Orientate Last Layer)
    [7 alg]
  3. PBL (Permutate Both Layer)
    [9 alg]

EG

EG consists of 3 steps:

  1. OFL (Orientate First Layer)
  2. There's 3 options, depending on the bottom face permutation:
    • if bottom face is solved: CLL
      [42 alg]
    • if bottom face need a T perm: EG-1
      [42 alg]
    • if bottom face need a Y perm: EG-2
      [42 alg]
